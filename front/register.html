Advanced Security Features

Strong Password Policy

Enforces complex passwords with uppercase, lowercase, numbers, and special characters to minimize unauthorized access risks.

Real-Time Password Strength

reCAPTCHA Protection

Prevents bots and automated attacks using Google reCAPTCHA to ensure only legitimate users can access the system.

Two-Factor Authentication (2FA)

Adds an extra layer of security by requiring OTP verification via email or authenticator apps during login.

Best Password Practices

  • Password must be at least 8 characters
  • Include uppercase letter
  • Include number
  • Include special character
  • Avoid characters: ( ) " :
  • Use unique email addresses
